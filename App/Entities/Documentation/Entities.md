
# Class vs Interface

## Key Differences

### Implementation
Interfaces
- define structure

classes
- define structure and implementation

### Inheritance:
Interfaces
- support multiple inheritance

classes 
- support single inheritance

### Static Members:
Interfaces 
- cannot have static members

classes 
- can have static members

### Runtime Presence:
Interfaces 
- do not exist at runtime

classes 
- do exist at runtime

### Use Cases:
Interfaces for 
- type-checking
- contracts

classes for 
- creating objects with behavior and state
